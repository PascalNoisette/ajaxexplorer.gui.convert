This plugin provides an entry in the context menu to convert a file into an other format.

Developper Note The only format implemented is tiff to pdf, feel free to create new adapters, this is a free software, by adding an entry in the plugin manifest (registry_contributions/convertions) referencing your worker class. Install note : - tiff to pdf It require the backend command tiff2pdf, please install it `sudo apt-get install libtiff-tools`
